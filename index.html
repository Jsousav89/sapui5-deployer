Welcome to SAPUI5-Deployer

SAPUI5-Deployer is a solution for deploying SAPUI5 and OpenUI5 web apps to a SAP ABAP Web Application Server automatically. It is designed to simplify the developer workflow by automating the following steps.

  • Download of Github Archive
  • Remove wrapper subdirectory from archive
  • Minify CSS, Javascript & XML files
  • Build Component-preload.js file where required
  • Recursively perform above steps for all Github submodules
  • Build complete new archive ready for deployment
  • Save deployment archive to file system
  • Deploy to SAP Internet Communications Framework
  • Flush cache

System Requirements

This solution was built on a SAP WAS ABAP 7.31 SP5 system with the SAP UI5 and SAP UI2 software components also installed. There is no reason it should not work on this version and all later releases. Earlier releases will need to be tested for compatibility.

The solution also makes use of the SAP provided function module /ui5/ui5_repository_load_http.

Whilst the installation instructions suggest copying this function module and changing it to support the direct input of the Github archive it is possible that SAP will modify this function module to support this in a future release.
